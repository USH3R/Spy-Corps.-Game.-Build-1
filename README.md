# **SPY CORPS**
A video game.
  
**Instructions to use Browser-based game**  
- If using a desktop computer or laptop:
1. In the repository, click on the large green button titled "<> Code".  
2. In the drop down menu shown, you will see the tabs "Local" and "Codespaces".
3. In the "Codespaces" tab, click on "Download ZIP" and
4. When the sub-window pops up instructing you to download the ZIP file,
5. Instruct the Save-to window where to save the ZIP file, and then
6. Click the "Save" button at the bottom right of that window. Then,
7. Go to the area on your computer where you saved the ZIP file.
8. Open the ZIP file,
9. Extract or move the file you find in the ZIP file.
10. Open the extracted file.
11. Click on the file called "index" or "index.html",
12. This will open the game in a browser.
13. Play the base game using arrow keys on your keyboard.

Displays a player (red square)  
Allows movement:  
⬆️ Up  
⬇️ Down  
⬅️ Left  
➡️ Right

Visual feedback  
  
Everything else (enemies, maps, weapons, SpyCore systems) can be built on top of this.
